Etiquetas de HTML

Etiqueta <!DOCTYPE html>

Declara el tipo de documento y la versión de HTML que se utiliza.

Ejemplo:

<!DOCTYPE html>

Etiqueta <html>

Es la etiqueta raíz del documento HTML y contiene todas las demás etiquetas.

Ejemplo:

<html lang="es"></html>

Etiqueta <head>

Contiene información sobre el documento, como el título, los metadatos y los enlaces a archivos CSS o JavaScript.

Ejemplo:

<head><title>Mi Página</title></head>

Etiqueta <title>

Establece el título del documento, que se muestra en la barra de título del navegador.

Ejemplo:

<title>Título de la página</title>

Etiqueta <body>

Contiene el contenido del documento, como texto, imágenes, enlaces, formularios, etc.

Ejemplo:

<body><p>Contenido de la página</p></body>

Etiqueta <h1> - <h6>

Definen los títulos de diferentes niveles, siendo <h1> el título más importante.

Ejemplo:

<h1>Encabezado 1</h1>

<h2>Encabezado 2</h2>

Etiqueta <p>

Define un párrafo de texto.

Ejemplo:

<p>Este es un párrafo.</p>

Etiqueta <a>

Define un enlace a otra página web o a un correo electrónico.

Ejemplo:

<a href="https://www.ejemplo.com">Enlace</a>

Etiqueta <img>

Inserta una imagen en el documento.

Ejemplo:

<img src="imagen.jpg" alt="Descripción">

Etiqueta <ul> y <ol>

Definen listas no ordenadas y ordenadas, respectivamente.

Ejemplo:

<ul><li>Elemento</li></ul>

<ol><li>Elemento</li></ol>

Etiqueta <li>

Define un elemento de una lista.

Ejemplo:

<li>Elemento</li>

Etiqueta <table>

Define una tabla de datos.

Ejemplo:

<table><tr><td>Dato</td></tr></table>

Etiqueta <tr>

Define una fila de una tabla.

Ejemplo:

<tr><td>Dato</td></tr>

Etiqueta <td>

Define una celda de una tabla.

Ejemplo:

<td>Dato</td>

Etiqueta <form>

Define un formulario para recopilar datos del usuario.

Ejemplo:

<form><input type="text"></form>

Etiqueta <input>

Define un campo de entrada de datos en un formulario.

Ejemplo:

<input type="text">

Etiqueta <select>

Define un menú desplegable en un formulario.

Ejemplo:

<select><option>Opción</option></select>

Etiqueta <textarea>

Define un área de texto en un formulario.

Ejemplo:

<textarea>Texto</textarea>

Etiqueta <button>

Define un botón en un formulario.

Ejemplo:

<button>Botón</button>

Etiqueta <div>

Define un contenedor genérico para otros elementos.

Ejemplo:<div>Contenido</div>

Etiqueta <span>

Define un contenedor para texto o elementos en línea.

Ejemplo:<span>Texto</span>

Etiqueta <header>

Define la cabecera de un documento o sección.

Ejemplo:<header>Cabecera</header>

Etiqueta <nav>

Define una sección de navegación.

Ejemplo:<nav><a href="#">Enlace</a></nav>

Etiqueta <main>

Define la sección principal de un documento.

Ejemplo:<main>Contenido principal</main>

Etiqueta <section>

Define una sección de un documento.

Ejemplo:<section>Sección</section>

Etiqueta <article>

Define un artículo o contenido independiente.

Ejemplo:<article>Artículo</article>

Etiqueta <aside>

Define un contenido relacionado con el contenido principal.

Ejemplo:<aside>Contenido relacionado</aside>

Etiqueta <footer>

Define el pie de página de un documento o sección.

Ejemplo:<footer>Pie de página</footer>

Etiqueta <figure>

Define un contenido que se puede utilizar como referencia.

Ejemplo:<figure><img src="imagen.jpg" alt="Descripción"><figcaption>Descripción de la imagen</figcaption></figure>

Etiqueta <figcaption>

Define la descripción de un contenido.

Ejemplo:<figcaption>Descripción</figcaption>

Etiqueta <details>

Define un contenido que se puede mostrar u ocultar.

Ejemplo:<details><summary>Detalles</summary><p>Contenido detallado</p></details>

Etiqueta <summary>

Define el resumen de un contenido.

Ejemplo:<summary>Resumen</summary>

Etiqueta <datalist>

Define una lista de opciones para un campo de entrada.

Ejemplo:<input list="opciones"><datalist id="opciones"><option value="Opción 1"></datalist>

Etiqueta <output>

Define el resultado de una operación.

Ejemplo:<output name="resultado"></output>

Etiqueta <progress>

Define la barra de progreso de una operación.

Ejemplo:<progress value="50" max="100"></progress>

Etiqueta <meter>

Define una barra de medición.

Ejemplo:<meter value="0.6"></meter>

Etiqueta <abbr>

Define una abreviatura.

Ejemplo:<abbr title="Abreviatura">ABR</abbr>

Etiqueta <address>

Define la información de contacto de un autor o una organización.

Ejemplo:<address>Dirección</address>

Etiqueta <area>

Define un área de una imagen que se puede utilizar como enlace.

Ejemplo:<map name="mapa"><area shape="rect" coords="0,0,100,100" href="#"></map>

Etiqueta <audio>

Define un archivo de audio.

Ejemplo:<audio src="audio.mp3" controls></audio>

Etiqueta <b>

Define un texto en negrita.

Ejemplo:<b>Negrita</b>

Etiqueta <bdi>

Define un texto que se debe renderizar en una dirección diferente a la del texto circundante.

Ejemplo:<bdi>Texto</bdi>

Etiqueta <bdo>

Define la dirección del texto.

Ejemplo:<bdo dir="rtl">Texto</bdo>

Etiqueta <br>

Define un salto de línea.

Ejemplo:<br>

Etiqueta <canvas>

Define un área de dibujo en una página web.

Ejemplo:<canvas id="lienzo" width="200" height="100"></canvas>

Etiqueta <cite>

Define la cita de un texto.

Ejemplo:<cite>Título de la obra</cite>

Etiqueta <code>

Define un fragmento de código.

Ejemplo:<code>console.log("Hola");</code>

Etiqueta <data>

Define un fragmento de datos.

Ejemplo:<data value="123">Datos</data>

Etiqueta <dfn>

Define la definición de un término.

Ejemplo:<dfn>Término definido</dfn>

Etiqueta <dialog>

Define un diálogo.

Ejemplo:<dialog open><p>Diálogo abierto</p></dialog>

Etiqueta <embed>

Define un objeto incrustado.

Ejemplo:<embed src="archivo.pdf" type="application/pdf">

Etiqueta <fieldset>

Define un grupo de campos de un formulario.

Ejemplo:<fieldset><legend>Grupo</legend><input type="text"></fieldset>

Etiqueta <i>

Define un texto en cursiva.

Ejemplo:<i>Cursiva</i>

Etiqueta <iframe>

Define un marco incrustado.

Ejemplo:<iframe src="pagina.html"></iframe>

Etiqueta <ins>

Define un texto que se ha insertado.

Ejemplo:<ins>Texto insertado</ins>

Etiqueta <kbd>

Define un texto que se debe introducir mediante el teclado.

Ejemplo:<kbd>Ctrl+C</kbd>

Etiqueta <label>

Define una etiqueta para un campo de un formulario.

Ejemplo:<label for="nombre">Nombre:</label>

Etiqueta <legend>

Define una leyenda para un grupo de campos de un formulario.

Ejemplo:<legend>Leyenda</legend>

Etiqueta <link>

Define un enlace a un recurso externo.

Ejemplo:<link rel="stylesheet" href="estilos.css">

Etiqueta <map>

Define un mapa de imagen.

Ejemplo:<map name="mapa"><area shape="rect" coords="0,0,100,100" href="#"></map>

Etiqueta <mark>

Define un texto que se debe resaltar.

Ejemplo:<mark>Resaltado</mark>

Etiqueta <menu>

Define un menú de opciones.

Ejemplo:<menu><li><button>Opción</button></li></menu>

Etiqueta <menuitem>

Define un elemento de un menú.

Ejemplo:<menuitem label="Opción">

Etiqueta <meta>

Define un metadato para un documento.

Ejemplo:<meta charset="UTF-8">

Etiqueta <noscript>

Define un contenido que se debe mostrar cuando el navegador no admite JavaScript.

Ejemplo:<noscript>JavaScript no está disponible</noscript>

Etiqueta <object>

Define un objeto incrustado.

Ejemplo:<object data="archivo.swf" type="application/x-shockwave-flash"></object>

Etiqueta <optgroup>

Define un grupo de opciones en un menú desplegable.

Ejemplo:<optgroup label="Grupo"><option>Opción</option></optgroup>

Etiqueta <option>

Define una opción en un menú desplegable.

Ejemplo:<option>Opción</option>

Etiqueta <param>

Define un parámetro para un objeto incrustado.

Ejemplo:<param name="parametro" value="valor">

Etiqueta <picture>

Define un contenedor para varias fuentes de imagen.

Ejemplo:<picture><source srcset="imagen-grande.jpg" media="(min-width: 800px)"><img src="imagen-pequena.jpg" alt="Imagen"></picture>

Etiqueta <pre>

Define un bloque de texto preformateado.

Ejemplo:<pre>Texto preformateado</pre>

Etiqueta <q>

Define una cita en línea.

Ejemplo:<q>Cita</q>

Etiqueta <rb>

Define un texto que se debe renderizar en una dirección diferente a la del texto circundante.

Ejemplo:<rb>Texto</rb>

Etiqueta <rp>

Define un texto que se debe renderizar en una dirección diferente a la del texto circundante.

Ejemplo:<rp>Texto</rp>

Etiqueta <rt>

Define un texto que se debe renderizar en una dirección diferente a la del texto circundante.

Ejemplo:<rt>Texto</rt>

Etiqueta <rtc>

Define un texto que se debe renderizar en una dirección diferente a la del texto circundante.

Ejemplo:<rtc>Texto</rtc>

Etiqueta <ruby>

Define un texto que se debe renderizar en una dirección diferente a la del texto circundante.

Ejemplo:<ruby>Texto</ruby>

Etiqueta <s>

Define un texto que se debe mostrar con una línea que lo atraviesa.

Ejemplo:<s>Texto tachado</s>

Etiqueta <samp>

Define un fragmento de código de muestra.

Ejemplo:<samp>Salida de ejemplo</samp>

Etiqueta <script>

Define un bloque de código JavaScript.

Ejemplo:<script>console.log("Hola");</script>

Etiqueta <select>

Define un menú desplegable.

Ejemplo:<select><option>Opción</option></select>

Etiqueta <slot>

Define un contenedor para un fragmento de código.

Ejemplo:<slot name="nombre"></slot>

Etiqueta <small>

Define un texto que se debe mostrar en una fuente más pequeña.

Ejemplo:<small>Texto pequeño</small>

Etiqueta <source>

Define una fuente de multimedia.

Ejemplo:<source src="video.mp4" type="video/mp4">

Etiqueta <strong>

Define un texto que se debe mostrar en negrita.

Ejemplo:<strong>Texto en negrita</strong>

Etiqueta <style>

Define un bloque de código CSS.

Ejemplo:<style>p { color: red; }</style>

Etiqueta <sub>

Define un texto que se debe mostrar como subíndice.

Ejemplo:<sub>2</sub>

Etiqueta <sup>

Define un texto que se debe mostrar como superíndice.

Ejemplo:<sup>2</sup>

Etiqueta <table>

Define una tabla de datos.

Ejemplo:<table><tr><td>Dato</td></tr></table>

Etiqueta <tbody>

Define el cuerpo de una tabla.

Ejemplo:<tbody><tr><td>Dato</td></tr></tbody>

Etiqueta <td>

Define una celda de una tabla.

Ejemplo:<td>Dato</td>

Etiqueta <template>

Define un contenedor para un fragmento de código.

Ejemplo:<template><p>Contenido</p></template>

Etiqueta <textarea>

Define un área de texto en un formulario.

Ejemplo:<textarea>Texto</textarea>

Etiqueta <tfoot>

Define el pie de una tabla.

Ejemplo:<tfoot><tr><td>Pie</td></tr></tfoot>

Etiqueta <th>

Define una celda de encabezado en una tabla.

Ejemplo:<th>Encabezado</th>

Etiqueta <thead>

Define el encabezado de una tabla.

Ejemplo:<thead><tr><th>Encabezado</th></tr></thead>

Etiqueta <time>

Define una fecha y hora.

Ejemplo:<time datetime="2023-10-27">27 de octubre de 2023</time>

Etiqueta <tr>

Define una fila en una tabla.

Ejemplo:<tr><td>Dato</td></tr>

Etiqueta <track>

Define pistas de texto para elementos multimedia (como subtítulos).

Ejemplo:<track src="subtitulos.vtt" kind="subtitles" srclang="es" label="Español">

Etiqueta <u>

Define un texto que se debe mostrar con una línea subrayada.

Ejemplo:<u>Texto subrayado</u>

Etiqueta <ul>

Define una lista no ordenada.

Ejemplo:<ul><li>Elemento</li></ul>

Etiqueta <var>

Define una variable.

Ejemplo:<var>x</var>

Etiqueta <video>

Define un archivo de video.

Ejemplo:<video src="video.mp4" controls></video>

Etiqueta <wbr>

Define un punto de ruptura de línea.

Ejemplo:<p>Texto largo con<wbr>punto de ruptura</p>
